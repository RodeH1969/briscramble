📍 LOCATION ACCESS NEEDED
To race on BriScramble, GPS must be enabled.
Follow these steps on your iPhone:
1
Open iPhone Settings
2
Tap Privacy & Security → Location Services
3
Make sure Location Services is ON
4
Scroll down to Safari Websites → set to While Using
5
Return to Safari and tap the button again
⚠ Private/Incognito mode blocks GPS entirely.
Make sure you are using a normal Safari tab.
⚠ Does not work in Private / Incognito mode
